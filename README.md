## 🚀 О проекте

Это **чистый, современный фронтенд маркетплейса**, собранный на **React + Vite + ES6**.  
Не те старые костыли — современный стек, быстрый билд, адекватная архитектура.

**Что он умеет:**
- 📦 Просмотр товаров
- 🛒 Добавление в корзину
- 🔍 Поиск/фильтрация
- 💡 Динамичный UI без лагов
