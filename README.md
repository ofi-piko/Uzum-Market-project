# 🚀 Uzum Market Project

Практичный e-commerce проект с нормальной структурой.  
Без лишнего булшита — только код, логика и понятные пути.

Проект можно использовать как:
- основу под маркетплейс
- пет-проект
- шаблон для коммерческой разработки
- портфолио

---

## 🧠 О чём проект

**Uzum Market Project** — это фронтенд e-commerce приложения с базовой бизнес-логикой:
каталог товаров, компоненты, сервисы, API-слой и конфигурации.

Код написан так, чтобы:
- легко читать
- легко масштабировать
- не словить пиздец через месяц поддержки
